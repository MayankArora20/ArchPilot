About ArchPilot

What is ArchPilot?

ArchPilot is an intelligent architecture visualization and requirement engineering platform that helps developers and architects understand, design, and document software systems.

Features

  • Automated repository analysis and architecture visualization
  • AI-powered requirement engineering through interactive chat
  • PlantUML diagram generation and visualization
  • Project management and tracking
  • Light and dark theme support

How It Works

  1. Add a project by providing a Git repository URL or describing requirements
  2. ArchPilot analyzes your codebase or requirements
  3. View generated architecture diagrams in PlantUML format
  4. Refine requirements through AI-powered chat
  5. Track and manage multiple projects

Technology Stack

Built with Angular 21, TypeScript, and modern web technologies. Powered by PlantUML for diagram generation and AI for intelligent requirement analysis.
